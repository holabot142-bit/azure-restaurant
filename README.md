## التعديلات المطلوبة

### 1. الصور
- استبدل `dish-placeholder` و `gallery-item` بصور حقيقية
- الصور المقترحة: أطباق، صالة المطعم، الشيف، الإطلالة

### 2. Google Maps
- في `contact.html`، استبدل `map-placeholder` بكود Google Maps Embed

### 3. روابط التواصل
- حدّث روابط السوشيال ميديا في الفوتر

### 4. معلومات الاتصال
- حدّث: رقم الهاتف، الإيميل، العنوان

### 5. الخطوط
- يمكن استبدال Google Fonts بخطوط محلية
